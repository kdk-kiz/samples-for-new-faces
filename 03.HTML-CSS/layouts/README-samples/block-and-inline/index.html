インライン要素

”ライン”とは、文章を書くときに延々続く行のようなものを指す。
作文で使ったあの方眼紙で、右上から下まで続いて次の行頭にいってまた続くあの行を想像してほしい。

インライン要素には、spanタグ、aタグ、iタグなどの装飾タグや、
今ここに書いている文字の塊（W3Cの既定でText-nodeという）、
テキストブロックやボタンなどのコントロール、画像が含まれる。

基本的に２種類ある。このように、中にかこったコンテンツにまとわりつくよう囲むもの。
このタイプは、途中で改行があると、次の行頭まで”ワープ”するように折り返す。

例外の要素

いくつか例外がある。

ひとつはテーブル要素。 これらは、タグひとつひとつが特殊な扱いとなっている。
CSSの「display」プロパティは、インライン要素・ブロック要素などを任意に書き換えることができ、また、一部はどちらの要素にも属さない。
今このサンプルにあるのが display: flex; を指定したもの。
中に入ったすべてのブロック要素が、横並びになっている。
もう一つ、display: inline-blockも紹介する。
ブロック要素であるが、インライン要素のように最小限の横幅にとどまり、
かつ横に並ぼうとする。
